# AWS Project - Farrer Park Hospital Freelance Nurse Website

## Project setup

### Install dependencies

```bash
npm install
```

### Compiles and hot-reloads for development

```bash
npm run dev
```

### Compiles and minifies for production

```bash
npm run build
```

### Lints and fixes files

```bash
npm run lint
```
